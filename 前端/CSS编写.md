# `CSS`编写

## 基本语法

==选择器+{声明；声明；......}==

选择器就是`HTML`的标签，选择对指定的标签进行修改。后面的多条声明就是对标签内容具体哪些属性的修改。

```html
<style>
  /* 针对<p>标签的修改 */
  p {
    /* 设置字体颜色 */
    color: red;
    /* 设置字体大小 */
    font-size: 30px;
  }
</style>

<p>
  这是具体的内容
</p>
```

注意：

1. `CSS`语言一般写在`<style>`标签里。
2. `style` 标签可以放到页面任意位置，但一般放到 `head` 标签内。
3. `CSS` 使用` /* */ `作为注释。

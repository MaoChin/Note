## 1. 词法分析器(扫描器)

词法分析的任务：从左至右逐个字符地对源程序进行扫描，产生 一个个单词符号。

单词符号：

1. 基本字（保留字）
2. 标识符：变量名，函数名等
3. 常数
4. 运算符
5. 界符：各种标点符号，空格等

`flex`文件：`*.l`

```c
// flex(fast lex, scanner)文件内容结构（分3部分）
/* P1: declarations(定义段) */
%{
  
%}

%%
  /* P2: translation rules(规则段) */
%%

/* P3: auxiliary functions(用户辅助程序段，c函数)*/
```

用`flex`运行：`flex xxx.l`，生成 `xxx.c`文件，然后：

```shell
gcc -lfl xxx.c -o xxx
```

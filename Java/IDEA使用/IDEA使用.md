# IDEA使用

## 插件

### 1. smart tomcat

`tomcat`的插件。

使用时需要配置：`Dit Configurations`。主要是配置本地的`Tomcat`路径。

若不使用插件，每次更改代码都需要重新打包，并把`war`包拷贝到`Tomcat`目录下，比较麻烦；使用插件后只需要更改代码然后重新运行即可。

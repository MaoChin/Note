# 总体大纲

## 1. 投资管理基础

#### 1.1 财务报表（资产负债表，利润表，现金流量表）

1. 资产负债表
   1. ==资产（流动，非流动） = 负债（流动，非流动） + 所有者权益（股本，资本公积，盈余公积，未分配利润）==，所有者权益也称为净资产
   2. 资产负债表又叫做==第一会计报表==，记录某一个时间点（季末，半年末，年末）的资产负债情况
2. 利润表
   1. 别称：==损益表==
   2. 动态报告，反应==一定时期==的经营成果，揭示企业财务状况发生变动的==直接原因==
   3. 利润表的作用：评价企业业绩；解析企业获利能力高低的原因.....
3. 现金流量表
   1. 别称：账务状况变动表
   2. 反应==特定会计期间内==企业的现金增减变动情况
   3. 是以==收付实现制==为基础编制的。收付实现制（具体收到/支出钱）《---》权责发生制（事件发生，不一定收到/支出钱）
   4. 净现金流 = 经营活动产生的现金流 + 投资活动产生的现金流 + 筹资活动产生的现金流

#### 1.2 财务报表分析（流动性比率，财务杠杆比率，盈利能力比率，营运效率比率）

1. 流动性比率
   1. 流动比率 = 流动资产（货币资金+应收账款+应收票据+存货）/流动负债
   2. 速动比率 = （流动资产 - 存货） / 流动负债
2. 财务杠杆比率
   1. 资产负债率 = 负债 / 资产
   2. 权益乘数（杠杆比率） = 资产 / 所有者权益
   3. 负债权益比 = 负债 / 所有者权益
   4. 利息倍数 = EBTI（息税前利润） / 利息 = （净利润 + 税 + 利息 ）/ 利息
3. 盈利能力比率
   1. 销售利润率 = 净利润 / 销售收入。每单位销售收入所产生的利润
   2. 资产收益率 = 净利润 / 总资产。每单位资产所产生的利润
   3. 净资产收益率 = 净利润 / 所有者权益（净资产）。每单位 所有者权益（净资产） 所产生的利润。衡量企业最大化股东财富能力
4. 营运效率比率
   1. ==存货周转率== = 年销售==成本== / 年均存货，年均存货 = （年初存货 + 年末存货）/ 2 。存货周转率反应 ==存货的周转次数==
   2. 存货周转天数 = 365 / 存货周转率
   3. ==应收账款周转率== = 销售==收入== / 年均应收账款。应收账款周转率反应 ==应收账款的周转次数==
   4. 应收账款周转天数 = 365 / 应收账款周转率
   5. ==总资产周转率== = 年销售收入 / 年均总资产。总资产周转率反应 ==所有资产的使用效率==

### 2.2 权益类投资

1. ==股票==的特征：
   1. 收益性（==最基本==）
   2. 风险性
   3. 流动性
   4. 永久性
   5. 参与性
2. 股票的价值：票面价值（发行时），账面价值，清算价值，内在价值（未来收益的现值）
3. 股票的价格：理论价格，市场价格（受市场==供求关系==的影响）
4. ==可转换债券==：是一种债券，体现了==债权债务关系==，但是持有人可以选择转换成股票，就变成了所有权关系
5. 可转换债券的基本要素：
   1. 标的股票：转换的目标股票
   2. 转换期限：持有人最少持有债券==6个月==，期限后才可转换
   3. 票面利率
   4. 赎回条款（==发行人进行赎回==）
   5. 回售条款（==投资者/持有人进行回售==）
   6. 转换价格
   7. 转换比例（转换价格 * 转换比例 = 可转换债权面值） 
6. ==权证==的分类：
   1. 按基础资产来源：认股权证（股份公司发行，==增发新股==售予权证持有人）；备兑权证（投资银行发行，==认兑已经流通的而不是增发==的股票）
   2. 按行权时间：美式权证（可以在权证失效前任意交易日行权）；欧式权证（只能在失效日当日行权）；百慕大式权证（可以在失效日前一段规定的时间内行权）

### 2.3 股票分析方法

1. 基本分析方法
   1. 宏观经济分析：各种指标（GDP.....）；经济周期；宏观经济政策
   2. 行业分析
   3. 公司分析：内在价值（股票未来收益的现值），市场价格（供求关系决定）

2. 技术分析方法
   1. 通过股价==历史信息==预测未来趋势。


### 2.4 股票估值方法

1. 内在价值法：按照==未来现金流的贴现==对公司的的内在价值进行评估。
   1. 企业自由现金流贴现模型：营运现金流 - 运营资本和固定资产投资 = 企业自由现金流
   2. 股东自由现金流贴现模型：企业自由现金流 - 向外部融资者支付利息 = 股东自由现金流
   3. 股利贴现模型：股东自由现金流 - 再投资需要的留存收益 = 股利
2. 相对价值法：使用公司的市盈率，市净率，市售率，市现率等指标与其竞争公司作比较。
   1. 市盈率 = 每股市价 / 每股收益
   2. 市净率 = 每股市价 / 每股净资产
   3. 市现率 = 每股市价 / 每股现金流
   4. 市销率 = 每股市价 / 销售收入

## 3. 固定收益投资

#### 3.1 债券与债券市场

债券的分类：

1. 按发行主体划分：政府债券，企业债券，金融债券
2. 按偿还期限划分：短期债券（1年以下），中期债券（1~10年），长期债券（10年以上）
3. 按计息方式划分：固定利率债券，浮动利率债券（基准利率（变化的）+固定利差），零息债券
4. 按付息方式划分：息票债券，贴现债券

#### 3.2 债券的风险

1. 信用风险：违约，不兑现
2. 利率风险：债券价格和银行利率是负相关关系 
3. 通货膨胀风险：购买力下降
4. 流动性风险：变现困难
5. 再投资风险
6. 提前赎回风险

#### 3.3 债券价值分析

1. 零息债券内在价值公式：V  = M / （1 + R）^ t；V = M (1 - tr/360)
1. 固定利率债券内在价值公式：V = C/(1 + r)  +  C/(1 + r)^2  +  C/(1 + r)^3  + ...+  C/(1 + r)^n + M/(1 + r)^n
1. 统一公债内在价值公式：V = C / r
1. 利率期限结构：上升型，反转型，水平型，拱形
5. 信用利差：
   1. 信用利差随着期限增加而扩大
   2. 信用利差随着经济周期的扩展而缩小，随着经济周期的收缩而扩张
   3. 信用利差受经济预期的影响

#### 3.4 货币市场工具

1. 特点：流动性高，风险低，期限短（低于一年），大额交易，债务契约
2. 短期回购协议：==证券抵押贷款==，抵押品以国债为主。缺钱的：正回购方；借钱的：逆回购方
   1. 种类：按回购期限：1天（隔夜回购），2天.....
   2. 按==逆回购方（有钱的）==是否有权处置回购标的国债：质押式回购，买断式回购

#### 3.5 其他货币市场工具

1. 同业拆借：金融机构之间以货币借贷方式进行短期资金融通的行为，交易主体为大型金融机构
2. 银行承兑汇票：贴现，再贴现，转贴现
3. 商业票据：期限2~270天，面额大，利率低
4. 大额可转让定期存单：银行发行，固定期限，==可以在二级市场上转让==

## 4. 衍生工具

#### 4.1 衍生工具概述

1. 由基础资产构成或衍生而来的交易合约，基础资产称为 合约标的资产
2. 衍生工具的特点：跨期性，杠杆性，联动性（衍生工具的价格和标的资产强相关），不确定性/高风险性
3. 衍生工具的分类：
   1. 按照合约特点分类：远期合约（非标准化，约定），期货合约（标准化，签署），期权合约（赋予==期权买方==一定选择的权力），互换合约，结构化金融衍生工具（上面四个的组合）
   2. 按产品形态分类：独立衍生工具，嵌入式衍生工具

#### 4.2 远期合约与期货合约

1. 远期合约：交易双方约定 在未来的某一时间，按==约定的价格==交易标的资产
2. 远期合约的特点：非标准化，实物交割，灵活。效率低，流动性差，违约风险高
3. 期货合约：交易双方签署的在未来 某个确定的时间 按 ==确定的价格==交易标的资产
4. 期货交易制度：
   1. 保证金制度：5%~10%
   2. 盯市制度：每天结算
   3. 对冲平仓制度：大部分合约都通过对冲平仓结算
   4. 交割制度

#### 4.3 期权合约

1. 期权的分类：
   1. 按期权买方执行期权的时限：欧式期权（只有约定的那一天），美式期权（一段时间）

#### 4.4 互换合约

1. 交易双方互换等值的标的资产

## 8. 投资交易管理

   2. 做市商可以给市场提供流动性，维持证券价格稳定
2. 指令驱动市场
   1. 核心是买方下达购买指令，买方下达卖出指令
   2. ==指令是核心==
   3. 成交原则：价格优先-》时间优先

#### 11.2 场内证券交易结算原则与方式

1. 结算原则：

   3. 货银对付原则：证券和资金==一手交钱，一手交货==
   4. 分级结算原则：结算机构负责==机构和结算参与人==之间的清算交收；结算参与人负责==结算参与人和客户==之间的清算交收
2. 结算方式
   1. 净额结算：又叫差额清算，==记录应收应付款项相抵（轧差）后的净额==

      2. 多边净额结算：多方轧差
   2. 全额结算：==逐笔金额结算==，对每笔交易独立结算，==不做轧差处理==

# 总体大纲

### 1. 金融市场，资产管理与投资基金

#### 1.1 金融与居民理财

金融：货币资金的融通。货币资金来源于==居民==（包括个人和企业）从事的生产活动。

==居民==是社会最古老，最基本的经济主体。目前，居民理财最主要的方式是==货币储蓄==和==投资==。

#### 1.2 金融市场

1. 分类：==资本市场（>1年），货币市场（<=1年）==；发行市场（一级市场），流通市场（二级市场）；现货市场（2个工作日内交割），期货市场；
2. 金融市场的构成要素：参与主体，金融工具，金融交易的组织方式
3. 参与主体：政府，中央银行，金融机构，企业居民（==个人居民==是主要的资金供给者）
4. 金融交易的组织方式：场外交易，场内交易，网络

#### 1.3 金融资产与资产管理

1. 金融资产的分类：==股权类==（股票），==债权类==（票据，债券）
2. 资产管理：参与者（委托方，受托方）；受托资产（金融资产）；管理方式
3. 风险与收益并存。卖者有责，买者自负。

#### 1.4 中国资产管理行业的状况

1. 基金管理公司：
   1. 公募：向不特定对象募集资金（特定对象要累计==超过200人==），
   2. 特定客户：
2. 私募机构：
   1. 合格投资者：单位净资产>=1000W；个人金融资产>=300W，或近三年年均收入>=50W

#### 1.5 投资基金

1. 投资基金是一种==组合投资，专业管理，利益共享（投资人之间），风险共担（投资人之间）==的集合投资方式。
2. 分类：
   1. 募集方式：公募基金，私募基金；
   2. 运作方式：开放式基金，封闭式基金；
   3. 法律形式：公司型，契约型，有限合伙形
   4. 投资对象：传统，另类

### 2. 证券投资基金

#### 2.1 概念

1. 别称：共同基金（美国），单位信托基金（英国，香港），集合投资基金（欧洲），证券投资信托基金（日本）
2. 特点：
   1. 集合投资，专业管理
   2. 组合投资，分散风险
   3. 利益共享，风险共担
   4. 严格监管，信息透明
   5. 独立托管，保障安全

#### 2.2 基金和其他金融产品的对比

1. 基金和股票，债券：
   1. 经济关系不同：股票（所有权关系）；债券（债务关系）；基金（信托关系）
   2. 收益与风险不同
   3. 资金投向不同
2. 基金和银行存款的差异：性质不同，收益与风险特性不同，信息披露程度不同
3. 基金和保险的差异：目的不同，对投资者要求不同，变现能力不同

#### 2.3 证券投资基金的运作和参与主体

1. 基金当事人：
   1. 基金份额持有人，有权利：开会，分钱，捍卫权力.....
   2. 基金管理人（基金管理公司），在==基金运作中具有核心作用==。
   3. 基金托管人（独立于管理人，商业银行，金融机构）。职责：资产保管，清算，监督....
2. 基金市场服务机构：基金管理人，基金托管人，销售机构，支付机构，登记机构.....
3. 基金监管机构和自律组织：中国证监会--对服务机构进行监督管理；自律组织---基金业协会，证券交易所

#### 2.4 证券投资基金的法律形式和运作方式

1. 根据法律形式划分：契约型基金（根据==基金合同==设立），公司型基金（根据==公司章程==设立）。不同点：
   1. 基金营运依据
   2. 法律主体资格
   3. 基金组织方式
   4. 投资者地位
2. 根据运作方式划分：
   1. 开放式基金（主流）：份额不固定；存续期限不确定；可以申购赎回，一般不上市交易；交易价格按照 基金每日单位资产净值；每日披露；
   2. 封闭式基金：份额固定；存续期限确定；期限内不能申购赎回，可以在二级市场交易；交易价格根据市场行情变化；每周披露；

#### 2.5 证券投资基金的起源和发展

1. 1868英国（世界第一支证券投资基金），1924美国（最早的==开放式==证券投资基金）
2. 两部重要法律：1940投资公司法，1940投资顾问法
3. 特点：美国主导；==集中趋势突出==......

### 3. 基金的类型

#### 3.1 基金的分类

1. 分类的意义
   1. 投资者
   2. 基金管理公司
   3. 监管部门
   4. 基金的==分类是进行基金评级的基础。==
2. 根据==投资对象==分类：股票基金（超过80%）；债券基金（超过80%）；货币基金（100%）；基金中基金（超过80%）；混合基金；另类投资基金
3. 根据==投资目标==分类：增长型基金；收入型基金；平衡型基金
4. 根据==投资理念==分类：主动型基金；被动型（指数）基金
5. 根据==基金的资金来源和用途==分类：在岸基金；离岸基金；国际基金
6. 特殊类型基金：避险策略基金；上市开放式基金（LOF）；分级基金

#### 3.2 股票基金

1. 按投资市场分类：国内股票基金；国外股票基金；全球股票基金
2. 按股票分类：
   1. 按股票 投资市值 大小：绝对规模（<5亿--小盘；5亿~20亿--中盘；>20亿--大盘）；相对规模（20%以下--小盘；20%~50%--中盘；50%以上--大盘）
   2. 按股票的投资规模：小盘，中盘，大盘
   3. 按投资股票的性质：价值型；平衡型；成长型（高风险，高回报）；

#### 3.3 债券基金

1. 债券基金和 （单一）债券 的区别：
   1. 收益不同
   2. 收益率不同
   3. 到期日不同
   4. 利率风险不同（单一债券随着到期日的临近，利率风险==下降==）
   5. 信用风险不同

#### 3.4 货币市场基金

1. 货币市场工具：到期日不足一年的短期金融工具。禁止投资的对象：股票；可转债；浮动利率债券；信用等级在AA+以下的债券																				
2. 发行机构：政府，金融机构，大型工商企业
3. 特定：流动性好，风险低，收益低

#### 3.5 混合基金

1. 类型：偏股型（50%~70%）；偏债型；股债平衡型（40%~60%）；灵活配置型

#### 3.6 避险策略基金

1. 本质是混合基金，尽量保本，同时追求一定的收益；前身是保本基金。在==投资到期时==保证本金。
2. 投资策略：
   1. 对冲保险策略；
   2. 固定比例投资组合保险策略（CPPI）：==价值底线，安全垫==
3. 避险策略基金的类型：==本金==保证；==收益==保证（我国没有这个）；==红利==保证

#### 3.7 交易型开放式指数基金（ETF）

1. 特点：
   1. ==被动操作==的指数基金
   2. 独特的==实物==（股票）申购赎回机制
   3. ==一级市场和二级市场==的交易制度（一级市场申购赎回，最小30W/50W/100W；二级市场买卖）。两个市场，所以存在套利
   4. 套利：
      1. 折价套利：二级市场价格低于一级市场
      2. 溢价套利：二级市场价格高于一级市场

#### 3.8 QDII基金

1. ==合格境内机构投资者==：境内募集资金，在境外投资。
2. QDII可以以 ==人民币，美元或其他主要外汇货币== 为计价货币募集。
3. QDII禁止投资：
   1. 不动产
   2. 房地产抵押按揭
   3. 贵金属或代表贵金属的凭证
   4. 实物商品
   5. 借入现金不能超过10%
   6. 利用融资购买证券

#### 3.9 分级基金

1. 一只基金，多类份额（母基金，A类份额，B类份额....），合并运作
2. 分级基金==可以在交易所上市交易==
3. 内含衍生工具和杠杆特性
4. 多种收益实现方式

#### 3.10 基金中基金（FOF）

1. 将==80%==以上的资产投资到==其他证券投资基金。==
2. ==“一篮子”，分散化，波动相对偏低==
3. 规范：
   1. 投资一只基金不能超过总资产的==20%==
   2. 不能投资具有衍生性质的产品

### 4. 证券投资基金的监管（重点）

#### 4.1 基金监管概述

1. 基金监管机构：
   1. 广义：有法定监管权的==行政机构，基金行业自律组织，基金机构内部监管部门，社会力量==等。
   2. 狭义：行政机构（最权威，最有效）
2. 监管的目标：
   1. ==保护投资者（首要目标）==
   2. ==规范证券投资活动（直接目标）==
   3. 促进投资基金和资本市场健康发展（重要目标）
3. 监管原则：
   1. 保障投资人利益
   2. 高效
   3. 审慎
   4. 适度
   5. 依法监管
   6. 公开，公平，公正（执法）

#### 4.2 中国证监会

1. 中国证监会担负国家对证券市场实施==集中统一监管==的职责。
2. 监管措施：
   1. ==检查（事中监管）==
   2. ==调查取证（基础，保障）==
   3. 限制交易（限制期限不得超过==15==个交易日，复杂的可以==延长15个==交易日）
   4. 行政处罚 

#### 4.3 基金业协会

1. 性质：基金业协会是证券投资基金行业的==自律性组织==，是==社会团体法人==
2. 组成：
   1. 观察会员（观察期一年）：私募基金的管理人，托管人
   2. 普通会员：公募基金的 基金管理人，基金托管人
   3. 联席会员：基金服务机构
   4. 特别会员：证券交易所，期货交易所，登记结算机构
3. 组织结构：
   1. ==权力机构：会员大会==
   2. ==执行机构：理事会==
4. 职责：
   1. ==会员==----教育和组织，自律，培训......
   2. 依法办理==非公开募集基金的登记备案==（公募基金是中国证监会）

#### 4.4 对基金管理人和基金托管人的监管

1. 基金管理人的职责：

   1. 募集，发售，登记；
   2. 基金备案手续；
   3. 对基金进行管理，记账，分配
   4. 披露......

2. 基金管理人职责终止：

   1. 被依法取消资格
   2. 被持有人大会解任
   3. 依法撤销；依法宣告破产
   4. 其他

   5. 基金职业道德（重点）

3. 当基金管理人职责终止，但是其管理的基金合同没有到期，==基金份额持有人大会==应当在==6个月==内选出新的，在选出新的管理人之前有==中国证监会==指定临时管理人。

4. 基金托管人资格的审核：

   1. 商业银行：由==中国证监会和中国银监会==核准
   2. 其他金融机构：由中国证监会核准

5. 基金托管人的职责：

   1. 保管
   2. 开设资金账户
   3. 保存记录，账册，报表等
   4. 出具意见，复核，审核
   5. 监督

6. 基金托管人的职责终止和基金管理人一样！！

#### 4.5 对公募基金的监管

1. 成为基金管理人需要向==中国证监会==进行==注册==（证监会需要在==六个月==内做出决定），注册需要提交以下文件：
   1. 申请报告
   2. 基金合同草案
   3. 基金托管协议草案
   4. 招募说明书草案
   5. 法律意见书
   6. 其他文件
2. 公募基金发售的条件
   1. 基金管理人在证监会进行注册
   2. 基金管理人或者销售机构办理发售
   3. 发售日的==3日前==公布相关文件
3. 公募基金的募集时间：在收到证监会准予注册的==六个月内==进行募集。超过六个月后再募集，若注册事项未发生实质性的改变，报证监会备案；否则重新注册。
4. 公募基金的募集期限：==自基金份额发售之日起三个月内。==
5. 募集要求：
   1. 开放式：超过==最低募集份额总额==，==份额 >= 2亿份，金额 >= 2亿元，人数  >= 200人==
   2. 封闭式：达到注册规模的==80%==，==人数  >= 200人==
6. 募集期满后==10日内==聘请法定验资机构进行验资，收到验资报告后==10日内==向证监会申请。
7. 募集失败：基金管理人的==固有资产==承担债务和费用，并在==30日内==返还投资人的钱，加上同期存款利息。

#### 4.6 基金份额持有人

1. 基金份额持有人大会的权利：
   1. ==决定==基金扩募或延长期限
   2. ==决定修改==基金合同或提前终止基金合同
   3. ==决定更换==基金管理人，基金托管人
   4. ==决定调整==基金管理人和托管人的报酬
   5. 其他
2. 基金份额持有人大会的召集：日常机构；基金管理人；基金托管人；持有份额==10%==以上的份额持有人
3. 基金份额持有人大会的召开：需要==50%以上==份额持有人参加；若不足，则可以在==原时间三个月后六个月之前==再次召开，这时只需要==1/3以上==的份额持有人参加即可。
4. 基金份额持有人大会表决：
   1. 每==一基金份额代表一票==
   2. 可以委托代理人表决
   3. 审议普通事项需要==1/2以上==
   4. ==转换基金运作方式，更换基金管理人或托管人，提前终止基金合同，与其他基金合并==等重要事项需要==2/3以上==表决
   5. 表决结果需要报证监会==备案，并公告==

#### 4.7 对私募基金的监管

1. 采取==登记制度==，私募基金的基金管理人向==基金业协会==登记。基金业协会需要在==20个工作日内==给出结果，==办结登记手续。==
2. 合格投资者（私募，==少于200==人）：单位的净资产>=1000万；个人的净资产>=300万或者 最近3年年收入大于50万；
3. 私募基金也需要托管人，==但是在基金合同中另有约定的除外！==

### 5. 基金职业道德

1. 道德的特征：==差异性，继承性，约束性，具体性==    技术差距
2. 职业道德的特征：==特殊性，继承性，规范性，具体性==     特技规矩
3. 基金职业道德：
   1. ==守法合规（最基础）==：调整 基金从业人员，基金行业 和 基金监管 之间的关系
   1. ==诚实守信（核心）==：调整各种人际关系的基本准则
   1. 专业审慎：调整基金从业人员 和 职业 之间的关系
   1. 客户至上：调整基金从业人员 和 投资人 之间的关系
   1. 忠诚尽责：调整基金从业人员 和 其所在机构 之间的关系
   1. 保守秘密

### 6. 基金的募集，交易与登记

#### 6.1 基金的募集

1. 公募基金：注册制
   1. 注册又分为 简易程序（证监会不超过==20个工作日==内审核） 和 普通程序（不超过==6个月==；==分级基金，基金中基金==）。

#### 6.2 基金的认购

1. 开放式基金的认购：
   1. 填写认购申请表；==全额==缴款；==已经正式受理的认购申请不得撤销==。
   2. 认购时按==金额==认购，募集结束后再将金额换算成份额。
   3. 认购手续费：有两种收费方式，认购时收取（低一些）和赎回时收取。==股票基金的手续费<=1.5%；债券基金的手续费<=1%，货币基金无手续费。==
2. 封闭式基金的认购：以==份额==为单位认购

#### 6.3 基金的交易，申购，赎回

1. 封闭式基金的==交易==（可以上市交易，封闭期内份额不变，不能申购赎回）：
   1. 上市交易前应由==基金管理人向证券交易所提出申请==。
   2. 封闭式基金上市交易的条件：基金==合同期限为5年以上==；募集==金额大于2亿元==；==份额持有人大于1000人==
   3. 发行结束后不能按基金净值买卖，而是在证券交易所（二级市场）==按市价买卖==
   4. 交易原则：==价格优先，时间优先==
   5. 交易费用：==不高于成交额的0.3%，起点5元==，由证券登记公司和证券公司平分
2. 开放式基金的==申购赎回==（不能上市交易）：
   1. 开放式基金认购，合同生效后有==最多3个月的封闭期==，而后可以申购赎回。
   2. ==金额认/申购，份额赎回。==
   3. 申购成立：投资人提交申购申请并付款
   4. 申购生效：基金份额登记机构==确认基金份额==
   5. 申购后==T+1日==办理登记手续，==T+2日==可赎回
   6. 赎回后 ==T+1==日办理扣除手续，自赎回之日起==7==个工作日内支付赎回款项

### 7. 基金的信息披露

#### 7.1 基金信息披露概述

1. 披露内容的基本原则：==真实性（最根本，最重要）；准确性；完整性；及时性；公平性==
2. 披露形式的原则：规范性；易（理）解性；易得性
3. ==巨潮资讯网==

#### 7.2 基金主要当事人的信息披露义务

1. 基金管理人：
   1. 基金发售==3日前==公告
   2. 基金合同==生效后的次日==公告
   3. ==合同生效后每6个月==结束之日起 45日内更新==招募说明书==
   4. 上市交易的（封闭式，ETF）：==上市交易后每日==开市前公告
   5. 基金==净值信息==的披露：==封闭式每周==公告一次；==开放式 认购期每周公告 资产净值和份额净值，申购期每个开放日次日披露 份额净值和累计份额净值==
2. 基金托管人：
   1. 基金发售==3日==前将 ==基金合同，托管协议公告在托管人网站==
   2. 基金==半年度报告==和==年度报告==中公告托管人报告
   3. 发生重大事件==2日内==进行公告：==托管主要人员一年内变动30%==；召开基金份额持有人大会（==30日前==公告）；.......

#### 7.3 基金募集期间涉及的披露文件

1. 基金合同：约定基金管理人，托管人，和份额持有人的权力义务。
2. 合同成立：投资者交钱；
3. 合同生效：基金募集结束向证监会提交验资报告并书面通过。
4. 基金招募说明书：基金管理人发布
5. 基金托管协议：==基金管理人和基金托管人==签订，明确双方的权力义务，保护基金份额持有人的合法权益

#### 7.4 基金净值公告

1. 开放
   1. 权益类
   2. 货币基金：==封闭期==收益公告，==开放日==收益公告，==节假日==收益公告。需要公告的：每万份收益，7日年化收益率
2. 封闭式

#### 7.5 基金定期报告

1. 季度报告
   1. 季度结束后==15个工作日==内编制，==不足两个月==的可以不编制
2. 半年报告
   1. 半年结束后的==60日内==编制
3. 年度报告
   1. 一年结束后==90日内==编制
   2. 要经过==2/3以上独立董事==签字同意，由==董事长签发==

#### 7.6 QDII基金的信息披露

2. QDII基金招募书中披露的内容：境外投资顾问，境外托管人信息，基金交易信息，风险信息

### 9. 基金的销售行为规范及信息管理（重点）

#### 9.1 基金宣传推介材料规范

1. 登载过往业绩：
   1. 6个月内的不登载
   2. 6个月<合同生效<1年：登载==合同生效之日==起的业绩
   3. 1年<合同生效<10年：登载==自合同生效当年开始==所有完整会计年度的业绩
   4. 合同生效>10年：==最近10个==完整会计年度

#### 9.2 基金经营机构客户信息保存期限

1. 金融机构（基金管理人，托管人）---==5年==
2. 基金销售机构---==15年==
3. 基金份额登记机构，证券期货经营机构---自基金账户销户之日起保存至少==20年==

#### 9.3 私募基金销售行为规范

1. 销售：私募基金管理人；在==中国证监会==注册取得基金销售业务资格且成为==中国基金业协会会员==
2. 私募基金产品风险等级评估：私募机构==自行评估==或者==委托第三方机构==进行评估。
3. 私募基金合同签署：
   1. 风险提示：在投资者签署基金合同==前==，募集机构进行风险提示，并==签署风险揭示书==。
   2. 资格审查：募集机构审查投资者的资产证明文件或收入证明
   3. 投资冷静期：给投资者设置==不少于24小时==的冷静期，期间募集机构==不能主动联系投资者==，冷静期自基金合同签署完毕且交纳认购款项后起算。
   4. 回访

# `git`常用指令

1. `git log`
2. `git reflog`
3. `git config`
4. `git diff`
5. `git status`
6. `git cat-file`

版本回退相关：

1. `git reset`
2. `git checkout --`（回退工作区）
3. `git rm`

分支相关：

1. `git branch`（查看分支）
2. `git branch [branchname]`（创建新分支）
3. `git checkout [branchname] ` （切换分支）
4. `git merge [branchname]`（合并分支到当前分支）
